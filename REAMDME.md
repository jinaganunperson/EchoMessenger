# (C# 코딩) 에코메신저
## 개요
- C# 프로그래밍 학습  
- 1줄 소개 : 사용자가 텍스트 박스에 텍스트를 입력하여 전송 버튼을 누르면 리스트 박스에 텍스트가 추가하는 프로그램.
- 사용한 플랫폼: -C#, .NET Windows Forms, Visual Studio, GitHub
- 핵심기능: 텍스트 박스에 텍스트를 입력한 후 전송 버튼을 누르면 리스트 박스의 텍스트가 리스트 박스로 전송됨.
- 화면구성: label(Echo Messenger의 이름), textbox(텍스트 입력 역할), listbox(텍스트 박스에 입력된 텍스트가 나타남), button(텍스트 박스에 전송 신호를 보냄)

## 배운내용
기본 ui 및 데이터를 연동하여 컨트롤 배치와 기본적인 속성을 제어하는 방법을 배움. 
버튼을 클릭할 시 텍스트박스의 텍스트를 리스트박스의 항목으로 추가하는 방법을 배움.
텍스트 박스 안에 타이핑 한 내용을 리스트 박스로 옮겨 넣는 방법을 배움
키보드의 엔터키 입력을 감지하여 버튼 클릭과 동일한 전송 로직이 실행되도록 하는 방법을 배움.
전송이 끝나면 입력창에 남겨진 기존 메시지를 삭제하는 방법을 배움
내용이 없는 빈 문자열이나 공백만 있을 때는 메시지가 전송되지 않도록 방지하는 방법을 배움.
