# (C# 코딩) 에코메신저
## 개요
- C# 프로그래밍 학습
- 1줄 소개 : 사용자가 텍스트 박스에 텍스트를 입력하여 전송 버튼을 누르면 리스트 박스에 텍스트가 추가하는 프로그램.
- 핵심기능: 텍스트 박스에 텍스트를 입력한 후 전송 버튼을 누르면 리스트 박스의 텍스트가 리스트 박스로 전송됨.
- 화면구성: label(Echo Messenger의 이름), textbox(텍스트 입력 역할), listbox(텍스트 박스에 입력된 텍스트가 나타남), button(텍스트 박스에 전송 신호를 보냄)
